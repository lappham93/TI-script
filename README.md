# Tax

### 1. Generate folder
```cmd
./generate_folder.sh [PROJECT] [YYYY]
```

### 2. Get source code

For example: nanboya project:
```cmd
./get_source_nanboya.sh [YYYY]
```

### 3. Description
- Get AOW and format it as "Item 1;Item 2;..."
- Run cmd
   ```cmd
   ./get_description.sh "Item 1;Item 2;..."
   ```
- Copy each part and paste into spreadsheet
